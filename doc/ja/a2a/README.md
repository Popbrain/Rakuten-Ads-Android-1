# App to App

## AppToApp SDKのインポート

以下二つを`build.gradle`のdependenciesに追加してください。
```gradle
  implementation 'com.rakuten.android.ads:rps:0.2.0'
  implementation 'com.rakuten.android.ads:rps-a2a:0.1.0'
```

## 実装

バナー広告と実装方法が変わらないため、[そちら](../bannerads/README.md)をご参照ください。
